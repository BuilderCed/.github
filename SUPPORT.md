# Support

## Getting Help

- **Issues**: Use the issue tracker in the relevant repository
- **Discussions**: For general questions, use GitHub Discussions (where enabled)
- **Security**: See [SECURITY.md](SECURITY.md) for vulnerability reports

## Response Times

This is a solo founder project. Response times vary:
- **Security issues**: Within 48 hours
- **Bug reports**: Within 1 week
- **Feature requests**: Reviewed monthly

## Before Opening an Issue

1. Search existing issues to avoid duplicates
2. Use the provided issue templates (bug report, feature request)
3. Include reproduction steps for bugs
